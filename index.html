Click Counter

This app will count the clicks the user make by clicking the "Click Me" button. By clicking the "Save" button, the user will save the number of clicks and it will count as round 1. Players can play for multiple rounds. It will also compute the total number of clicks for all rounds. When the user clicks on "Reset" button, it will erase all the number of saved clicks.


GOAL: Add a countdown timer when the user click on the "Click Me" button with the time set by the user.

Blackjack

User can play blackjack. User has the option to input their name or play as anonymous. Player can only play when their credit is greater than or equal to $50 which is the amount of credit to start the game. They can cash-in a minimum of $100. Players can start the game by click the "Start Game" button. When the game starts, players will bet additional $10 if they wish to get another card by pressing the "Get Card" button. They can request to get additional card while their total is below 22. By clicking the "New Game" button, the game will reset and the user can start another game.


GOAL: The top up amount will be added to the total credit after clicking the "Top Up" button. Player can't click the "New Card" button when their credit is below $10.


HTML, CSS, JavaScript
